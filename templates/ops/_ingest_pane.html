Automated

Runs the following steps automatically for all active companies with a configured scraper:

  1. Probe each company IR source for reachability
  2. Enqueue live IR scrape jobs (via configured scraper mode: RSS, index page, etc.)
  3. Fetch new EDGAR filings (8-K, 10-Q, 10-K, 20-F) for all companies with a CIK
  4. Ingest IR press releases from the local archive folder
  5. Run LLM + regex extraction on all unextracted documents

LLM-directed crawl is included when the checkbox below is ticked — it runs after EDGAR and IR ingest, before extraction. Leave it unchecked for faster runs when crawl-sourced documents are not needed. After this pipeline completes, review disagreements in the Review tab.
